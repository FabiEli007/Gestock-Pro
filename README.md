# -Gestock-Pro
🚀 Gestock Pro is a full-stack MERN inventory &amp; sales management system. It features 🔐 JWT authentication, 📦 product &amp; stock management, 💰 multi-product sales, and 📊 analytics dashboard. It also includes ☁️ DevOps with Docker &amp; Kubernetes (GKE) for scalable deployment, auto-scaling, and performance testing
